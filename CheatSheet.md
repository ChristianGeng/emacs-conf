## Basic Navigation 

C-p Previous line,   
C-b Backwaro  
Next line, C-n  
Forward, C-f  

C-v	Move forward one screenful  
M-v	Move backward one screenful  

M-< , beginning of file  
M-> , end of file  
C-h r emacs book  
C-h f	Describe a function  
C-h v	Describe variable  

C-l C-l put current pos to top of file
